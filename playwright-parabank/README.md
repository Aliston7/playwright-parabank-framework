# Playwright ParaBank Framework




Automation framework for ParaBank banking application using:

- Playwright
- JavaScript
- Page Object Model (POM)

## Features

- Valid login test
- Invalid login validation
- Empty credentials validation
- Logout functionality test
- Transfer funds test
- Negative transfer funds validation
- Page Object Model framework

## Project Structure

```bash
pages/
tests/
playwright.config.js
.env (not included in repository)
```

## Installation

```bash
npm install
```

## Run Tests

```bash
npx playwright test
```

## Run in Headed Mode

```bash
npx playwright test --headed
```
